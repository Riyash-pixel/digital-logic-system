# Lab 02 — Sequential Logic Design

## Overview
Designed and simulated sequential circuits including JK triggers and a universal shift register.

## Tools Used
- Lattice Diamond
- ModelSim / Questa

## Components
- Static JK trigger
- Master–Slave JK trigger
- Dynamic JK trigger
- 5-bit Universal Shift Register

## Process
- Derived control signals
- Designed circuits using logic elements
- Verified behavior through simulation

## Simulation
- Applied test signals
- Validated outputs using waveform analysis
