# Lab 01 — Combinational Logic Circuit Design

## Overview
Designed and implemented a combinational logic system using logic minimization and multiple implementation methods.

## Tools Used
- Lattice Diamond
- ModelSim

## Implementations
- AND-OR logic
- NAND logic (DeMorgan’s Law)
- MUX 4:1 design

## Process
- Input conversion and mapping
- Karnaugh map minimization
- Boolean equation derivation
- Circuit implementation under constraints

## Simulation
- Verified outputs using ModelSim
- Confirmed correctness across all implementations
