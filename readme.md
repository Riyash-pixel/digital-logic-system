# Digital Logic Systems — Design & Simulation

## Overview
This repository contains the design and simulation of combinational and sequential digital logic systems developed using Lattice Diamond and ModelSim.

## Contents

### Lab 01 — Combinational Circuits
- Karnaugh map minimization
- AND-OR implementation
- NAND implementation (DeMorgan’s Law)
- MUX 4:1 based design

### Lab 02 — Sequential Circuits
- JK triggers (Static, Master–Slave, Dynamic)
- 5-bit Universal Shift Register (asynchronous reset)

## Tools
- Lattice Diamond
- ModelSim / Questa

## Focus
- Logic design
- System-level understanding
- Simulation & verification
